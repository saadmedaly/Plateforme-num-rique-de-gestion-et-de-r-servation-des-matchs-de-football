## 🚀 كيفية التشغيل

### الطريقة السريعة (10 دقائق)

1. **تثبيت المكتبات**
```bash
cd matchi-web
npm install
```

2. **إعداد Supabase**
   - أنشئ مشروعاً على https://supabase.com
   - اذهب إلى SQL Editor
   - انسخ كود `supabase/schema.sql` وشغّله

3. **تحديث المتغيرات**
```bash
cp .env.local.example .env.local
```
ثم حدّث القيم في ملف `.env.local`:
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
   - `SUPABASE_SERVICE_ROLE_KEY`
   - `NEXTAUTH_SECRET`

4. **التشغيل**
```bash
npm run dev
```

5. **افتح المتصفح**
```
http://localhost:3000
```

🎉 **مبروك! الموقع يعمل!**
